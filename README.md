# CG10 造纸机烘缸群温度监控与蒸汽热力补偿系统

某大型造纸厂造纸机干燥部烘缸群表面瞬态温度红外扫描与蒸汽热力补偿系统。包含 Go 红外扫描探头客户端、Java Spring Boot 中央调度服务、Vue3 + ECharts 前端监控大屏。

## 模块说明

### 1. infrared-client (Go 红外扫描探头)
- 模拟 40 个烘缸，每个 64 个横向扫描点
- 100Hz 扫描频率，轮询扫描所有烘缸
- TCP 协议：`CG10`(4字节魔数) + 长度(4字节大端) + JSON Payload
- 内置三种故障仿真：左边缘低温、右边缘低温、中心凹陷
- 自动断线重连

### 2. tcp-server (Java Spring Boot 后端)
- **Netty TCP服务**：端口 9000，高性能接收 Go 客户端温度流
- **温度处理引擎**：
  - 8分区热传导分析算法
  - 三级异常检测 (info/warning/critical)
  - PID 阀门开度计算，秒级响应
- **InfluxDB 时序存储**：分区温度、采样点、统计指标批量写入
- **PLC 控制服务**：通过 Modbus TCP / OPC UA 下发蒸汽阀门指令（模拟实现）
- **WebSocket 广播**：100ms 间隔向前端推送实时热力图
- **REST API**：
  - `GET /api/heatmaps` - 所有烘缸热力图
  - `GET /api/heatmaps/{id}` - 单个烘缸详情
  - `GET /api/status` - 系统运行状态
  - `GET /api/valve-commands` - PLC补偿指令历史
  - `GET /api/valve-openings` - 当前阀门开度
  - `POST /api/valve-commands/manual` - 手动下发补偿指令

### 3. heatmap-frontend (Vue3 前端大屏)
- 深色科技感监控大屏界面
- 40 个烘缸状态总览网格（正常/预警/严重/离线）
- ECharts 2D 热力图展示单个烘缸表面温度分布
- WebSocket 实时数据推送
- PLC 蒸汽补偿指令日志面板
- 系统状态仪表盘（扫描速率、异常统计、温度趋势）

## 快速启动

### 环境要求
- Docker Desktop (运行 InfluxDB)
- Go 1.21+
- Java 17+ / Maven 3.8+
- Node.js 18+

### 一键启动（各开一个终端窗口）

```bash
# 终端 1: 启动 InfluxDB 时序数据库
./scripts/start-infra.sh

# 终端 2: 启动 Java 后端服务
./scripts/start-backend.sh

# 终端 3: 启动 Go 红外扫描模拟器
./scripts/start-client.sh

# 终端 4: 启动 Vue3 前端大屏
./scripts/start-frontend.sh
```

启动完成后访问：http://127.0.0.1:5173

### 故障仿真说明

Go 客户端启动后会自动注入三种故障：
- **第 8 秒**：#7 烘缸左侧边缘温度偏低 15°C（持续 30 秒）
- **第 18 秒**：#23 烘缸中心温度凹陷 10°C（持续 25 秒）
- **第 35 秒**：#35 烘缸右侧边缘温度偏低 15°C（持续 20 秒）

后端检测到异常后会自动计算阀门开度补偿，秒级下发 PLC 指令，可在前端大屏右侧日志面板实时观察。

## TCP 协议格式

```
+----------+----------+------------------+
|  CG10    |  Length  |   JSON Payload   |
| 4 bytes  | 4 bytes  |   Length bytes   |
+----------+----------+------------------+
```

JSON Payload 示例：
```json
{
  "ts": 1718409600000,
  "cid": 7,
  "temps": [105.3, 104.8, 90.2, 88.5, 92.1, ...]
}
```

## 温度分区算法

每个烘缸的 64 个扫描点划分为 8 个控制区域，对应 8 组独立蒸汽阀门：

| 区域 | 位置 | 对应阀门 |
|------|------|----------|
| Z0-Z1 | 左边缘 | 左侧进气阀组 |
| Z2-Z3 | 左中部 | 左中进气阀组 |
| Z4-Z5 | 右中部 | 右中进气阀组 |
| Z6-Z7 | 右边缘 | 右侧进气阀组 |

当某区域温度低于基准温度超过阈值时，系统按 PID 公式计算阀门开度补偿：
```
Δ开度 = 温度偏差 × 控制增益(0.8)
```
补偿范围：-15° ~ +15°，开度范围 0% ~ 100%。
